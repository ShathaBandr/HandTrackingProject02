# نظام التحكم بالماوس باستخدام إيماءات اليد

## تطوير: شذا

مشروع يعتمد على تقنيات الرؤية الحاسوبية (Computer Vision) يتيح للمستخدم التحكم في مؤشر الفأرة (Mouse) باستخدام إيماءات اليد بشكل مباشر عبر الكاميرا، بدون الحاجة لاستخدام الماوس التقليدي.

---

## المميزات

* التحكم بالمؤشر باستخدام حركة الإصبع
*  النقر (Click) باستخدام إيماءة الأصابع
*  النقر بالزر الأيمن (Right Click)
*  التمرير (Scroll) باستخدام حركة اليد
*  إيقاف التحكم (Pause) عند فتح اليد
*  استجابة لحظية (Real-Time)

---

## التقنيات المستخدمة

* Python
* OpenCV
* MediaPipe
* Autopy

---

## طريقة التشغيل

1. تثبيت المتطلبات:

```bash
pip install -r requirements.txt
```

2. تشغيل المشروع:

```bash
python main.py
```

---

## فكرة المشروع

يهدف هذا المشروع إلى تطوير طريقة تفاعلية حديثة للتحكم في الحاسوب باستخدام إيماءات اليد، مما يفتح المجال لتطبيقات متعددة مثل:

* العروض التقديمية
* التحكم بدون لمس (Touchless Systems)
* تطبيقات الواقع التفاعلي
